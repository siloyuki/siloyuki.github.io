## CTF/Wargame 카테고리란?

Capture The Flag 대회와 워게임 사이트(Dreamhack, pwnable.kr, HackTheBox 등)에서 풀어본 문제의 **풀이(Write-up)** 를 기록합니다.

## 다루는 분야

| 분야 | 설명 |
|------|------|
| **Pwn** | 바이너리 익스플로잇, BOF, ROP 등 |
| **Rev** | 리버싱, 정적/동적 분석 |
| **Web** | SQL Injection, XSS, SSRF 등 |
| **Misc** | 포렌식, 암호학, 기타 |

## 기록 방식

- 문제 요약 → 접근 방법 → 풀이 과정 → 익스플로잇 코드 → 회고
- 못 푼 문제도 **어디서 막혔는지** 기록합니다
