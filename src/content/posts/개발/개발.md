직접 개발했거나 개발 중인 프로젝트를 정리했다.

## Wakku Wakku

- 기술: React, TypeScript, Web
- 상태: 공개 완료
- 토스 앱 환경에서 즐길 수 있도록 만든 캐릭터 수집 및 미니게임 웹 앱.
- [플레이](https://siloyuki.github.io/wakku-wakku-preview/) / [GitHub](https://github.com/siloyuki/wakku-wakku)

## BeatGrid

- 기술: Unity, C#
- 상태: 개발 중
- 박자 단위 이동과 그리드 기반 패턴 회피를 결합한 Unity 2D PC 게임 프로토타입.
- [GitHub](https://github.com/siloyuki/BeatGrid)

## JJAPGEKI

- 기간: 2024.03 - 2024.09
- 기술: Unity, C#, Arduino
- 아케이드 리듬게임 O.N.G.E.K.I의 플레이 경험을 PC 환경에서 구현한 프로젝트.
- 아두이노 기반 전용 컨트롤러를 직접 설계하고 제작했다.

## FO Protect

- 기간: 2023.03 - 2023.09
- 기술: Unity, C#
- 웨이브 사이 인터벌에 장비 업그레이드와 스탯 강화를 진행하는 무한 웨이브 디펜스 게임.
